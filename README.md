# 🌱 Sensor Monitoring Project

This project integrates multiple sensors with a microcontroller to monitor environmental and water quality parameters.  

---

## 📊 Sensors Used (Total: 4)

1. **BME280 Probe** – Measures temperature, humidity, and pressure  
2. **Analog pH Sensor Kit** – Detects pH levels in water  
3. **JSN-SR04T V3.0** – Waterproof ultrasonic distance sensor  
4. **TDS Sensor** – Measures Total Dissolved Solids (water quality)

---

## 🔧 Connection & Sensor Verification

Before running the system, ensure:

- All sensors are properly connected to the microcontroller.  
- Power supply and wiring are stable.  
- Each sensor is tested individually to confirm correct readings.  

### ✅ Example: Checking TDS Sensor
Run a simple test script to verify the TDS sensor output. Ensure values change when immersing the probe in different water samples.
